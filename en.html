Visualization and Calculation System for Underfloor Heating Projects (MVP)

The project is designed to facilitate the work of suppliers of electric-water underfloor heating systems. The system allows users to draw the contour of a room on a convenient grid, calculates the dimensions of the room, and generates all possible combinations of heating elements within the working power range. The user can choose the most suitable combination, and a report will be generated with an image of the layout of heating elements for that combination and a list of all project components. The application is dockerized, information on how to run the local version is available in the repository.

Stack: FastAPI, Angular, MongoDB, Docker
